## Deploy to GitHub Pages (FREE live URL)

1. Go to **github.com** → New repository → name it `portfolio`
2. Upload ALL 13 files (keep them all in root folder, no subfolders)
3. Go to **Settings → Pages → Source: main branch / root**
4. Click Save → live at `https://yourusername.github.io/portfolio`

## Update your intro video
